# Sistema de Gestión de Clientes, Servicios y Reservas

Proyecto desarrollado en Python utilizando Programación Orientada a Objetos (POO) para la empresa Software FJ.

## Funcionalidades
- Registro de clientes
- Gestión de servicios
- Creación de reservas
- Manejo de excepciones
- Registro de errores en archivo log

## Tecnologías
- Python
